# Home Automation System Using Bluetooth (HC-05)

## Objective
To control home appliances remotely using a smartphone via Bluetooth communication.

## Components / Tools
- Arduino UNO R3
- Bluetooth module (HC-05)
- Relay module
- LEDs (as appliance indicators)
- Jumper wires and breadboard
- Smartphone with Bluetooth control app

## Features
- Control appliances ON/OFF using phone
- Connect multiple devices to Arduino via relays
- Real-time wireless control with HC-05
- Demonstrates basic IoT and home automation concepts

## How it Works
1. Arduino is connected to relays controlling appliances.
2. HC-05 Bluetooth module receives commands from the smartphone app.
3. Arduino executes commands to turn devices ON or OFF.
